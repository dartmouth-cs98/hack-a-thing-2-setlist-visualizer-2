Setlist Visualizer Tutorial

To choose your own concert/artists to visualize, first go to setlist.fm and find the desired artist. Then, follow the following sections to fill out the form on the main page

Unique Artist URL

This is the unique URL that setlist.fm uses for that artist. For example, if you were to Google search: "Chainsmokers setlist fm", you'd probably land here:

https://www.setlist.fm/setlists/the-chainsmokers-33db30f9.html

From this, the unique URL is:

the-chainsmokers-33db30f9.html

Start URL

What setlist do you want to start at? Say I have a particular timeframe in mind. Then, I would go to the first setlistfm URL I want. Let's start with Dave Matthew's concert at the Gorge:

daresbury-estate-halton-england-6be83e6e.html

Stop URL

What setlist do you want to end at? If you don't care and want the first 10 pages, then just type:

''''

Or provide a URL where the tour ended, or any concert you want to stop at, following the same process as above in the Start URL:

cluj-arena-cluj-napoca-romania-7beb7eb0.html
